Activity F: GROUP BY and HAVING Clause

You are working with a SALES table that contains sales records. Write a SQL query to:

Display the total sales for each product, but only include products where the total sales exceed $500.

Table Schema: 
SALES
-----
product_id (INT)
quantity (INT)
total_price (DECIMAL)

Hint:

Remember that:

  • GROUP BY is used to group rows that have the same values
  • HAVING is used to filter groups (similar to WHERE but for groups)
  • Aggregate functions like SUM() work with grouped data

Try writing a query that groups by product_id and sums the total_price, then filters for sums greater than 500.
