These questions test your conceptual understanding of SQL. Think carefully about each answer before checking the AI feedback.

Conceptual Questions

  1. What is the difference between a WHERE clause and a HAVING clause in a SELECT query?

    Consider these aspects in your answer: - When each clause is applied in query execution - The types of conditions they can filter - Their relationship with GROUP BY

  2. Explain why it might be appropriate to declare an attribute that contains only digits as character data type instead of numeric data type.

    Think about: - How the data will be used - Special cases like phone numbers or IDs - Leading zeros and formatting

  3. Explain the difference between an ORDER BY clause and a GROUP BY clause.

    Consider: - Their purposes in query processing - Effect on result set - Relationship with aggregate functions

  4. What is the difference between the COUNT aggregate function and the SUM aggregate function?

    Think about: - What each function calculates - The data they work with - NULL value handling

  5. Explain why it would be preferable to use a DATE data type to store dates instead of a character data type.

    Consider: - Data validation - Special date functions available - Sorting and comparison behavior

    Consider: - deleting table or tuples

  7. What is difference between logical operators of AND operator and OR operator?

    Consider: - values that must be true - whether all value must be true or only one of the two values has to be true
