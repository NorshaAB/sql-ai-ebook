# ✅ Solution for Exercise 3.1 – Basic JOIN Query

## 🔍 SQL Query

```sql
SELECT c.FirstName, c.LastName
FROM customers c
INNER JOIN invoices i ON c.CustomerId = i.CustomerId
WHERE i.Total > 100;
