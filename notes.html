SQL Quick Reference

üìö Data Definition Language (DDL)

Commands that define the database structure:

  • CREATE DATABASE - Creates a new database
  • CREATE TABLE - Creates a new table
  • DROP DATABASE - Deletes a database
  • DROP TABLE - Deletes a table
üí° DDL commands are used to create and modify the database structure itself.

üîç Data Manipulation Language (DML)

Commands that work with the data:

  • SELECT - Retrieves data from database
  • INSERT - Adds new records
  • UPDATE - Modifies existing records
  • DELETE - Removes records

Aggregate Functions

üìä Data Aggregation

Perform calculations across sets of rows:

COUNT()
MIN()
MAX()
SUM()
AVG()
-- Example: SELECT department, AVG(salary) FROM employees GROUP BY department HAVING AVG(salary) > 50000;
  • GROUP BY - Groups rows by specified columns
  • HAVING - Filters groups (like WHERE for GROUP BY)

Joins

üîó Outer Join

Returns all rows from one table and matching rows from another:

-- Left Outer Join (returns all left table rows) SELECT * FROM table1 LEFT JOIN table2 ON table1.id = table2.id;
‚ö†Ô∏è Without a join condition, you get a Cartesian product (all possible row combinations)!

ü§ù Natural Join
